자주 묻는 질문 (FAQ)

배처 툴(Batcher Tools)은 어떤 도구인가요?
배처 툴(Batcher Tools)은 대용량 SVG 벡터 이미지 변환(PNG/JPG), WAV/MP3 오디오 변환, 대량의 파일 이름 일괄 변경(Batch Rename) 기능을 웹 브라우저 내에서 쉽고 빠르게 처리해 주는 스마트 로컬 유틸리티 도구입니다. 복잡한 개발 환경 세팅이나 프로그램 설치 없이 크롬이나 웨일 등 브라우저에서 바로 작동하며, 다중 배율 내보내기, 오디오 음질 세부 조절, 다채로운 파일 이름 변경 규칙 적용 등 디자이너와 개발자들을 위한 유용한 편의 기능들을 지원합니다.
내가 입력한 파일이나 파일명이 외부 서버로 전송되나요?
아니요, 절대 전송되지 않습니다! 본 사이트는 최신 브라우저의 File System Access API 및 웹 이미지/오디오 디코딩 기술을 활용하여 모든 파일 읽기, 이미지 변환, 이름 변경 및 포맷 패키징 작업을 100% 사용자의 PC/웹 브라우저 내부(Client-Side)에서 처리합니다. 서버로 어떠한 파일 정보나 이미지/오디오 바이너리 데이터도 전송하지 않으므로 기밀 프로젝트 리소스나 개인 미디어 유출 걱정 없이 안전하게 사용하실 수 있습니다.
브라우저 호환성 및 출력 디렉토리(폴더) 기능은 어떻게 동작하나요?
Google Chrome, Microsoft Edge 등 최신 File System Access API를 지원하는 브라우저에서는 사용자가 연동한 로컬 원본 디렉토리 구조 그대로 변환된 파일을 직접 저장하거나 파일명을 실제 로컬에서 즉시 변경(Direct Write Mode)할 수 있습니다. Safari, Firefox 등 해당 API를 완벽히 지원하지 않는 브라우저의 경우, 브라우저가 제공하는 안전 메모리 환경 안에서 폴더 구조를 유지한 채 파일 처리를 마친 뒤 최종적으로 하나의 ZIP 압축 패키지로 자동 빌드되어 일괄 다운로드되도록 동작합니다.
각 탭별로 어떤 변환 설정과 변경 규칙을 사용할 수 있나요?
선택한 탭에 따라 각각 다른 세부 설정 패널이 제공됩니다.
- 이미지 변환 탭: 필요한 배율(1.0x, 1.5x, 2.0x)을 동시 활성화하고 파일명 끝에 붙을 접미사(예: @2x, @1.5x)를 커스텀하여 자동 적용할 수 있으며, PNG 또는 JPG 포맷으로 저장 가능합니다.
- 오디오 변환 탭: 원하는 출력 음질 비트레이트(128kbps, 192kbps, 256kbps, 320kbps)를 조절하고, 탐색할 입력 확장자 필터(.wav, .mp3 등)를 세부 선택할 수 있습니다.
- 이름 변경 탭: 특정 문자열 치환, 앞이름/뒷이름 붙이기, 특정 위치 문자 지우기, 숫자만 남기기, 괄호 내용 지우기, 순차적인 일련번호 매기기, 확장자 강제 일괄 변경/삭제 등 다양한 규칙 조합을 제공합니다. 더불어 대기열 목록 내의 이름을 더블클릭하여 직접 개별 수정하는 것도 가능합니다.
이름 변경 적용 전에 실수를 방지하거나 되돌릴 수 있나요?
이름 변경 탭에서는 규칙을 적용할 때마다 우측 대기열의 '변경할 파일명' 칸에 실시간으로 어떻게 변경될지 미리 시각적으로 보여줍니다. 최종적으로 '이름 변경 적용' 버튼을 누르기 전에는 실제 로컬 파일명이 바뀌지 않으므로 안전하며, 규칙을 적용하는 도중 실수했다면 '실행 취소(Undo)' 버튼을 눌러 이전 상태의 임시 이름으로 되돌리거나 '이름 복원(Reset)'을 눌러 최초의 원래 이름으로 한 번에 초기화할 수 있습니다.
'원본 파일 자동 정리' 또는 '선택 파일 삭제' 시 주의할 점은 무엇인가요?
이미지/오디오 변환 탭에서 제공되는 '원본 파일 자동 정리'는 변환 작업 성공 시 불필요해진 이전 포맷 원본 파일을 로컬 디렉토리에서 자동으로 안전하게 영구 삭제해 주는 편의 기능입니다. 또한, 이름 변경 탭의 '선택 파일 삭제'는 연동된 폴더 내의 특정 파일들을 일괄적으로 삭제하는 기능입니다. 해당 동작들은 로컬 디스크에서 영구 삭제되어 되돌릴 수 없으므로 중요 원본 데이터는 작업 전에 반드시 사전에 복사해 두거나 백업해 두시는 것을 강력히 권장합니다.
변환되거나 변경된 파일들을 상업적 용도로 배포/사용해도 되나요?
네, 전적으로 가능합니다. 배처 툴 자체와 내부 동작에 이용되는 각종 오픈소스 모듈들은 MIT, BSD, SIL 등 저작권 제약이 거의 없는 무료 라이선스로 구성되어 있습니다. 변환되거나 이름이 변경된 결과물 파일에 어떠한 소유권이나 권리 제약도 설정하지 않으므로 개인적 이용은 물론이고, 상업용 소프트웨어 개발 리소스나 유료 디자인 팩, 비즈니스 콘텐츠 제작 및 배포 등 모든 산업 영역에서 제약 없이 자유롭게 활용하실 수 있습니다.

Frequently Asked Questions (FAQ)

What kind of tool is Batcher Tools?
Batcher Tools is a smart, local utility tool designed to batch convert SVG images (to PNG/JPG), batch convert audio formats (WAV/MP3 to MP3), and bulk rename large sets of files directly inside the web browser. It works immediately in modern browsers without requiring any installation or software setup and includes multi-scale scaling options, audio bitrate adjustment, and powerful filename editing rules.
Are my files or filenames uploaded to external servers?
No, never! This tool leverages the browser's File System Access API and client-side rasterization/decoding capabilities to read, process, and download files 100% locally inside your web browser (Client-Side). No file buffers, audio/image binary data, or filenames are ever sent to an external server. You can safely use it for confidential or proprietary project assets.
How do browser compatibility and local directory modes work?
Modern browsers like Google Chrome and Microsoft Edge that support the WICG File System Access API can save converted images directly into your local directory or rename local files instantly. For browsers that do not support this API (such as Safari or Firefox), file processing occurs in browser-protected memory, and output results are packaged into a standard ZIP archive for batch downloading, retaining your original folder layout.
What options and renaming rules are available in each tab?
Settings panels adapt depending on the active tab:
- SVG Images Tab: Toggle multiple output scale ratios (1.0x, 1.5x, 2.0x), set custom suffixes (e.g. @2x, @1.5x), and export as PNG or JPG format.
- Audio Tab: Adjust the encoding bitrate (128kbps, 192kbps, 256kbps, 320kbps) and specify the target extensions to scan for (.wav, .mp3, etc.).
- Rename Tab: Apply rules like replacing text, adding prefixes/suffixes, cutting character ranges, keeping numbers only, stripping bracketed content, applying auto-incremental serial numbers, and modifying extensions. You can also **double-click any filename** in the queue to edit it manually.
Can I undo changes or prevent mistakes when bulk renaming?
Yes. The rename tab shows a real-time preview of the target filenames under the "New Name" column as you add rules. No changes are written to the actual files until you click "Apply Rename." If you apply a rule by mistake, you can use the **"Undo"** button to revert the last rule or **"Reset"** to restore all names to their original filenames instantly.
What precautions apply for 'Clean up original files' or 'Delete Selected'?
The 'Clean up original files' option automatically deletes the original SVG or WAV files after successful conversion. The 'Delete Selected' button in the rename tab physically deletes chosen files from your directory. Because these operations permanently erase files on your disk, **they cannot be undone. We strongly advise backing up important files** before utilizing these features.
Can I use the processed files for commercial purposes?
Yes, completely! Batcher Tools and its core conversion algorithms are built using free, open-source packages (such as MIT, BSD, SIL licenses) with no usage restrictions. We impose no copyright terms or licenses on your converted or renamed output files, so you can freely distribute and use them for commercial projects, design bundles, software builds, or paywalled content.
