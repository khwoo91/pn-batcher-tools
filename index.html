자주 묻는 질문 (FAQ)

Batcher는 어떤 도구인가요?
Batcher는 대용량 SVG 벡터 이미지나 WAV 오디오 파일 등 다양한 파일들을 브라우저 내에서 쉽고 빠르게 일괄 변환(Batch Convert)해 주는 스마트 로컬 유틸리티 도구입니다. 복잡한 개발 환경 세팅이나 프로그램 설치 없이 웹 브라우저에서 바로 작동하며, 다중 배율 내보내기 및 음질 세부 조절 등 디자이너와 개발자들을 위한 유용한 편의 기능들을 다양하게 지원합니다.
내가 입력한 파일들이 외부 서버로 유출되거나 전송되나요?
아니요, 절대 전송되지 않습니다! 본 사이트는 최신 브라우저의 File System Access API 및 웹 래스터라이징/오디오 디코딩 모듈을 이용하여 모든 파일 읽기, 변환 처리, 다운로드 패키징 프로세스를 100% 사용자의 웹 브라우저(Client-Side) 안에서 로컬로 처리합니다. 어떠한 네트워크 전송도 수행하지 않으므로 기밀 프로젝트 리소스나 개인 미디어 유출 걱정 없이 안심하고 사용하실 수 있습니다.
브라우저 호환성 및 출력 디렉토리(폴더) 기능은 어떻게 동작하나요?
Google Chrome, Microsoft Edge 등 최신 WICG File System Access API를 지원하는 브라우저에서는 사용자의 원본 로컬 디렉토리 구조 그대로 이미지를 내보내거나 기존 파일을 정리하는 모드를 지원합니다. Safari, Firefox 등 해당 API를 미지원하는 브라우저의 경우에는 브라우저 안전 환경 안에서 폴더 구조를 유지한 채 ZIP 압축 패키지로 자동 빌드되어 일괄 다운로드하도록 처리됩니다.
출력 하위 폴더명을 입력하지 않고 비워두면 어떻게 되나요?
출력 하위 폴더명을 지정하지 않으면, 각 원본 파일이 들어있는 로컬 원본 서브디렉토리 경로 내에 직접 변환 파일을 생성합니다. 이 방식을 사용하면 기존의 폴더 구조를 그대로 유지하면서 원본 파일 바로 옆에 변환 완료된 이미나 음원을 깔끔하게 배치할 수 있습니다.
변환 옵션 및 파일명 접미사는 어떻게 조정하나요?
선택한 탭에 따라 각각 다른 세부 설정 패널이 제공됩니다. 이미지 탭에서는 필요한 배율(1.0x, 1.5x, 2.0x)을 활성화하고 파일명 끝에 붙을 접미사(예: @2x, _hd)를 직접 수정하여 자동 적용할 수 있습니다. 오디오 탭에서는 원하는 출력 비트레이트(예: 128kbps, 192kbps, 320kbps)를 자유롭게 조절하여 인코딩 품질을 설정할 수 있습니다.
'원본 파일 자동 정리' 옵션을 활성화하면 어떻게 되나요?
변환 프로세스가 성공적으로 끝나면, 변환에 사용되었던 기존 로컬 디렉토리 내의 원본 파일들(.svg, .wav 등)을 자동으로 안전하게 삭제해 줍니다. 변환 작업 완료 후 불필요해진 원본 파일을 자동으로 비워주는 편리한 기능이지만, 삭제된 원본 파일은 다시 복구할 수 없으므로 중요한 원본 데이터는 사전에 반드시 백업해 두시는 것을 적극 권장합니다. (ZIP 압축 다운로드 모드에서는 지원하지 않는 로컬 전용 실험적 기능입니다.)
변환 포맷에 따라 파일 내용(배경 투명도 등)은 어떻게 처리되나요?
이미지(PNG/JPG): PNG 포맷 선택 시 원본의 투명 채널(알파 채널)을 손실 없이 그대로 보존하여 내보냅니다. JPG 포맷 선택 시에는 투명 처리된 배경 영역을 자동으로 흰색(#FFFFFF) 배경으로 대체하여 출력합니다.
오디오(WAV/MP3): 고용량 WAV 오디오 파일을 브라우저 내에서 빠르고 안전하게 디코딩한 뒤 대중적인 MP3 오디오 파일 포맷으로 일괄 변환하여 인코딩합니다.
변환된 파일 결과물을 상업적으로 마음대로 사용해도 되나요?
네, 물론입니다. 본 도구 자체와 내부 변환 모듈은 MIT, BSD-3-Clause, SIL OFL 1.1 등 제약이 없는 완전 자유 오픈소스 라이선스 패키지로 빌드되었습니다. 따라서 변환 결과물에 어떠한 라이선스 제약도 부과하지 않으며 개인 포트폴리오는 물론, 유료 판매용 디자인 패키지나 상업용 앱 빌드 리소스 등 모든 비즈니스 환경에서 자유롭게 활용 및 배포하실 수 있습니다.

Frequently Asked Questions (FAQ)

What kind of tool is Batcher?
Batcher is a smart local utility tool that quickly batch converts large volumes of files (such as SVG vector images and WAV audio files) directly in the browser. It operates completely in your browser without requiring complex dev environments or software installations and supports diverse convenient features for designers and developers.
Are my files sent to any external server?
No, never! This site utilizes the latest browser File System Access API along with client-side rasterization and audio decoding modules to process all file reading, conversion, and downloading 100% locally inside your web browser (Client-Side). No network transfer is performed, so you can safely use it without worrying about leaks of confidential projects or personal media assets.
How do browser compatibility and output directory features work?
Modern browsers like Google Chrome and Microsoft Edge that support the WICG File System Access API can export files directly into your original local directories, maintaining directory structure. For browsers that do not support this API (such as Safari or Firefox), files are automatically zipped in a virtual archive and offered as a single ZIP download, keeping the folder structure intact.
What happens if I leave the Output Subfolder field empty?
If left blank, the converted files will be **generated directly within the original subdirectory where each source file is located**. This preserves your directory structure, placing converted files right next to their corresponding source files.
How do I adjust conversion settings and filename suffixes?
Different configuration options are provided depending on the active tab. For **Images**, you can toggle scale options (1.0x, 1.5x, 2.0x), select the target format (PNG/JPG), and customize filename suffixes. For **Audio**, you can select the target output bitrate (e.g., 128kbps, 192kbps, 320kbps) to control encoding quality.
What does the "Clean up original files" option do?
Once the export is successful, it will **automatically delete the original files (.svg, .wav, etc.) from your local directories**. This is convenient for organizing your files post-conversion, but since this action is permanent and irreversible, we **strongly recommend backing up important data** beforehand. (This experimental local-only feature is not supported in the ZIP download fallback mode.)
How are files processed during conversion (transparency, formatting, etc.)?
**Images (PNG/JPG)**: Choosing PNG preserves the alpha channel (transparency) of your original SVG without any loss. Choosing JPG automatically fills transparent background areas with solid white (#FFFFFF).
**Audio (WAV/MP3)**: WAV audio files are decoded in-browser and converted into standard MP3 format with your chosen bitrate.
Can I use the converted files for commercial purposes?
Yes, absolutely! This tool and its internal conversion modules are built on fully open-source packages (MIT, BSD-3-Clause, SIL OFL 1.1) that have no commercial restrictions. We impose no licensing terms on your outputs. You are free to use and distribute them in any environment, including personal portfolios, commercial design kits, or paid application assets.
