자주 묻는 질문 (FAQ)

svg-Batcher는 어떤 도구인가요?
svg-Batcher는 대용량 SVG 벡터 이미지 파일을 브라우저 상에서 원하는 배율과 포맷(PNG, JPG)으로 빠르게 일괄 변환(Batch Export)해 주는 스마트 로컬 유틸리티 도구입니다.
내가 입력한 SVG 파일이 외부 서버로 유출되거나 전송되나요?
아니요, 절대 전송되지 않습니다! 본 사이트는 최신 브라우저의 File System Access API 및 가상 ZIP 라이브러리를 이용하여 모든 파일 읽기, 이미지 래스터라이징, 다운로드 빌드 프로세스를 100% 사용자의 웹 브라우저(Client-Side) 안에서 로컬로 처리합니다. 어떠한 네트워크 전송도 수행하지 않으므로 기밀이나 상업적 디자인 리소스 유출 걱정 없이 안심하고 사용하실 수 있습니다.
브라우저 호환성 및 출력 디렉토리(폴더) 기능은 어떻게 동작하나요?
Google Chrome, Microsoft Edge 등 최신 WICG File System Access API를 지원하는 브라우저에서는 사용자의 원본 로컬 디렉토리 구조 그대로 이미지를 내보내거나 기존 파일을 정리하는 모드를 지원합니다. Safari, Firefox 등 해당 API를 미지원하는 브라우저의 경우에는 브라우저 안전 환경 안에서 폴더 구조를 유지한 채 ZIP 압축 패키지로 자동 빌드되어 일괄 다운로드하도록 처리됩니다.
변환된 이미지 결과물을 상업적으로 마음대로 사용해도 되나요?
네, 물론입니다. 본 도구 자체와 내부 변환 모듈은 MIT, BSD-3-Clause, SIL OFL 1.1 등 제약이 없는 완전 자유 오픈소스 라이선스 패키지로 빌드되었습니다. 따라서 변환 결과물에 어떠한 라이선스 제약도 부과하지 않으며 개인 포트폴리오는 물론, 유료 판매용 디자인 패키지나 상업용 앱 빌드 리소스 등 모든 비즈니스 환경에서 자유롭게 활용 및 배포하실 수 있습니다.

Frequently Asked Questions (FAQ)

What kind of tool is svg-Batcher?
svg-Batcher is a smart local utility tool that quickly batch converts (Batch Export) large volumes of SVG vector image files into desired scales and formats (PNG, JPG) directly in the browser.
Are my uploaded SVG files sent to any external server?
No, never! This site utilizes the latest browser File System Access API and virtual ZIP library to process all file reading, image rasterization, and download building 100% locally inside your web browser (Client-Side). No network transfer is performed, so you can safely use it without worrying about leaks of confidential or commercial design resources.
How do browser compatibility and output directory features work?
Modern browsers like Google Chrome and Microsoft Edge that support the WICG File System Access API can export images directly into your original local directories, maintaining directory structure and optionally cleaning up original files. For browsers that do not support this API (such as Safari or Firefox), files are automatically zipped in a virtual archive and offered as a single ZIP download, keeping the folder structure intact.
Can I use the converted images for commercial purposes?
Yes, absolutely! This tool and its internal conversion modules are built on fully open-source packages (MIT, BSD-3-Clause, SIL OFL 1.1) that have no commercial restrictions. We impose no licensing terms on your outputs. You are free to use and distribute them in any environment, including personal portfolios, commercial design kits, or paid application assets.
