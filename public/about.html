서비스 소개 About Batcher Tools

웹 브라우저 기반의 로컬 다기능 일괄 처리 유틸리티 Browser-based local multi-functional bulk processing utility

배처 툴(Batcher Tools)은 대량의 SVG 이미지 변환, 고해상도 오디오 인코딩 및 일괄적인 파일명 규칙 재정의를 소프트웨어 설치 없이 웹 브라우저 내에서 직접 수행할 수 있도록 설계된 차세대 스마트 로컬 웹 유틸리티 도구입니다.

무거운 로컬 변환 소프트웨어를 설치하거나 민감한 자료를 검증되지 않은 불투명한 외부 웹 서버로 전송할 필요 없이, 크롬(Chrome) 및 엣지(Edge) 등 표준 브라우저 API가 제공하는 강력한 샌드박스 내부 보안 구조를 기반으로 작동합니다.

완전무결한 로컬 브라우저 구동 (Server-less & Zero Upload): 배처 툴은 어떠한 원본 파일 정보, 이미지/오디오 바이너리, 혹은 수정된 파일명 목록 정보도 절대 서버로 전송하거나 외부에 보관하지 않습니다. 모든 처리 작업은 오직 사용자의 로컬 환경 내 브라우저 힙(Heap) 메모리와 로컬 CPU 자원만을 활용하여 동작하므로, 저작권이 중요하거나 기밀에 속하는 프로젝트 소스 리소스를 마음 편히 다룰 수 있습니다.

제공하는 핵심 기능

SVG 대량 이미지 변환

벡터 포맷인 SVG 파일을 1.0배수, 1.5배수, 2.0배수 등 고배율의 고해상도 PNG 또는 JPG 포맷으로 순식간에 일괄 추출하여 내보냅니다. UI/UX 디자인 에셋 대량 출력 작업에 적합합니다.

WAV/MP3 오디오 일괄 인코딩

용량이 매우 큰 무압축 WAV 오디오 파일을 브라우저 내부 오디오 코덱을 활용해 대중적이고 압축률이 높은 MP3 파일로 즉시 변환합니다. 128kbps부터 320kbps까지 원하는 음질을 직접 선택하여 제어할 수 있습니다.

대량 파일 이름 일괄 변경 (Batch Rename)

특정 문자 찾아서 바꾸기, 앞/뒤에 문자 추가, 자리수 기준 삭제, 공백 지우기, 일련번호 매기기, 괄호 안 텍스트 소거 등 현업에서 가장 많이 필요한 일괄 네이밍 규칙 시스템을 실시간 프리뷰와 실행 취소(Undo)를 지원하며 자유자재로 설정할 수 있습니다.

로컬 디렉토리 직접 기록 (WICG API)

최신 브라우저가 제공하는 File System Access API를 지원하여, 웹 브라우저에서도 사용자가 권한을 수락한 실제 로컬 디렉토리 경로에 파일을 즉각 덮어쓰거나 직접 작성(Direct Output)할 수 있어 별도로 매번 다운로드받아 정리하는 불편함을 해소했습니다.

Batcher Tools is a next-generation smart local web utility designed to perform bulk SVG image conversion, high-resolution audio encoding, and batch filename rule redefinition directly inside your web browser without requiring any software installation.

Instead of installing heavy local conversion software or sending sensitive files to unverified, non-transparent external web servers, it operates based on a robust sandbox security structure provided by standard modern browsers such as Chrome and Edge.

100% Local Browser Execution (Server-less & Zero Upload): Batcher Tools never uploads or stores any of your source files, image/audio binary data, or modified filename lists to any external server. All processing tasks are performed entirely inside your local browser's Heap memory using only your local CPU/RAM resources, allowing you to process copyrighted or highly confidential project source resources with complete peace of mind.

Core Features

SVG Bulk Image Conversion

Batch export SVG vector files into high-resolution PNG or JPG formats at multiple scale ratios (1.0x, 1.5x, 2.0x) instantly. Ideal for batch-exporting UI/UX design assets.

WAV/MP3 Audio Encoding

Batch convert large uncompressed WAV files into compact, widely compatible MP3 files using browser-native codecs. Customize output quality with target bitrates from 128kbps to 320kbps.

Bulk Rename (Batch Rename)

Apply a combination of rules—such as replacing text, adding prefixes/suffixes, cutting character ranges, removing brackets, keeping numbers, and auto-numbering—with a real-time preview and safe Undo/Reset options.

Direct Local Directory Access (WICG API)

Leveraging the File System Access API in modern browsers, you can write converted or renamed files directly to your local directories without the hassle of downloading and unpacking ZIP files.

Development Philosophy

  • Privacy First: We process everything client-side to fundamentally eliminate data leak risks.
  • No Installation: Get ready in 1 second by opening your browser—no OS restrictions.
  • Free and Commercial Use: No usage limits or premium paywalls. All output files are yours to use commercially or non-commercially without copyright restrictions.
